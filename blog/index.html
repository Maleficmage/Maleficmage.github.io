Navigation Bar

Navigation bar inspiration:
YouTube = https://www.youtube.com/watch?v=biOMz4puGt8
GitHub = https://github.com/fireship-io/222-responsive-icon-nav-css

The NavBar is used to provide the client with access to different pages on the site from the sides of the screen. The biggest eyesore when using this method of navigation is the scrollbar, body::-webkit-scrollbar has options to customize the site scrollbar to look more in line with the navbar. The Icons were created from fontawesome.com and require inserting the fontawesome source reference in the head of the document. Once the source is referenced you use an SVG element with a nested g element to group and give the image properties. then, use a path to reference the icon in your group. By adding classes to the path you can transition the color when the user hovers over the icon. Creating CSS classes for logos and ':last-child' there can be extensive customization for the 'home' and last elements.
